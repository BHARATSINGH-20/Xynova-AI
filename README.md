# 🤖 XYNOVA AI

XYNOVA AI is a full-stack AI assistant web application built using **React (Frontend)** and **FastAPI (Python Backend)** powered by the **Groq AI API**.

It allows users to interact with an AI assistant directly from a web interface.

---

## 🚀 Features

- AI Chat Assistant
- Fast responses using Groq AI
- Modern React UI
- FastAPI backend
- REST API communication
- Simple and clean interface

---

## 🛠 Tech Stack

### Frontend
- React
- JavaScript
- Tailwind CSS
- Axios

### Backend
- Python
- FastAPI
- Uvicorn

### AI Integration
- Groq API
- LLaMA Model
